  ├───────┼────────────────────────────────────┤
  │ → / ← │ Seek +/- 5 seconds                 │
  ├───────┼────────────────────────────────────┤
  │ . / , │ (Paused) Step 1 frame forward/back │
  ├───────┼────────────────────────────────────┤
  │ + / - │ Speed up / slow down (0.25x to 4x) │
  ├───────┼────────────────────────────────────┤
  │ u     │ Undo last mark                     │
  ├───────┼────────────────────────────────────┤
  │ q     │ Save and quit                      │
  ├───────┼────────────────────────────────────┤
  │ Esc   │ Quit without saving                │
  └───────┴────────────────────────────────────┘

  What you see on screen

  - Top bar — timestamp, frame number, speed, pause indicator
  - Timeline bar — red (violence) / green (normal) colored segments with a white playback cursor
  - Bottom bar — current label in color, plus controls hint
